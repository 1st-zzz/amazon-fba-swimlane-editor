亚马逊 FBA 泳道图 - 在线文档

版本说明：适用于当前编辑器（支持行列编辑、并行组、累计时间、备注框）。

快速访问

先启动本地服务，再打开下方地址：

http://localhost:8765/canvases/amazon-fba-swimlane-editor.html

打开泳道图编辑器 跳转到编辑区

若页面未更新，请按 Ctrl+Shift+R 强制刷新。

核心操作

  • 行（泳道）：点击左侧泳道名选中，双击改名，拖拽上下排序。
  • 列（阶段）：点击列标题选中，双击改名，拖拽左右排序。
  • 节点：拖拽改行列，双击改名；顶部可编辑 ID、用时、并行组、备注、下游。
  • 备注框：节点可填备注，图中显示黄色备注框。
  • 列整理：点击工具栏 整理列，可按列内容重命名、排序并删除空列。

时间与并行规则

  • 底部时间支持小时/天口径（如 8小时、3天）。
  • 并行组（同 parallelGroup）按 最长用时 合并，不做求和累计。
  • 同列展示取该列最长累计节点作为代表，避免多条时间干扰阅读。

数据与导入导出

  • 保存：按钮 保存 会写入浏览器本地存储。
  • 恢复默认：按钮 恢复默认 清理当前编辑状态。
  • 导出/导入：通过 JSON 文件跨环境迁移流程配置。

本地服务启动命令

在项目根目录执行：

python -m http.server 8765

或直接双击 canvases/start-editor-server.bat。
